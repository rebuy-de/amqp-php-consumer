amqp-php-consumer
=================

This is the documentation for the `amqp-php-consumer library <https://github.com/rebuy-de/amqp-php-consumer>`_.

This library allows you to define consumers for AMQP with `php attributes`_.
For consuming messages the AMQP-Library `videlalvaro/php-amqplib`_ is used.
